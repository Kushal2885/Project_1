# Household Socio-Economic Data Analysis

---

# Project Overview

This project focuses on analyzing a socio-economic household dataset using Python and statistical techniques. The objective is to understand household demographics, income distribution, education levels, and statistical properties such as skewness, kurtosis, and dispersion.

## The project includes:
- Theory concepts of statistics
- Dataset creation
- Data analysis using Python
- Visualization and interpretation
- Statistical distribution analysis

---

# Objective

To analyze household socio-economic data and apply statistical concepts including:

- Types of Data
- Descriptive Statistics
- Measures of Central Tendency
- Measures of Dispersion
- Gaussian Distribution
- Skewness and Kurtosis
- Data Visualization

# Key Findings

- Household income shows slight positive skewness.
- Graduate and Post-Graduate households generally earn higher incomes.
- Income distribution varies significantly across education levels.
- Most households belong to middle-income category.
- Standard deviation indicates noticeable variation in income.
